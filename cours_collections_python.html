01 — list [ ]

La Liste

La liste est la collection la plus utilisée en Python. Elle stocke une séquence ordonnée d'éléments, qui peuvent être de types différents. Elle est mutable : on peut la modifier après sa création.

Création et accès

# Création
fruits = ["pomme", "banane", "cerise", "mangue"]
notes  = [12, 15.5, 9, 18, 14]
mixte  = ["Alice", 25, True, 3.14]   # types mixtes OK
vide   = []                          # liste vide

# Accès par index (commence à 0)
print(fruits[0])      # "pomme"   — premier
print(fruits[2])      # "cerise"
print(fruits[-1])     # "mangue"  — dernier
print(fruits[-2])     # "cerise"  — avant-dernier

# Slicing [début:fin:pas]  (fin exclus)
print(fruits[1:3])    # ["banane","cerise"]
print(fruits[:2])     # ["pomme","banane"]
print(fruits[2:])     # ["cerise","mangue"]
print(fruits[::2])    # ["pomme","cerise"]   — 1 sur 2
print(fruits[::-1])   # inversée

Itération et compréhensions de liste

# Boucle simple
for fruit in fruits:
    print(fruit)

# Avec index : enumerate()
for i, fruit in enumerate(fruits):
    print(f"{i}: {fruit}")

# List comprehension — syntaxe puissante !
carres    = [x**2 for x in range(10)]
pairs     = [x for x in range(20) if x % 2 == 0]
majuscule = [f.upper() for f in fruits]
admis     = [n for n in notes if n >= 10]

# Fonctions utiles sur les listes numériques
print(sum(notes))    # total
print(min(notes))    # minimum
print(max(notes))    # maximum
moyenne = sum(notes) / len(notes)
⚠️

Attention aux copies ! b = a ne copie pas la liste, elle crée un alias. Modifier b modifie aussi a. Utilisez b = a.copy() ou b = a[:] pour une vraie copie.

02 — tuple ( )

Le Tuple

Le tuple est une séquence ordonnée et immuable. Contrairement à la liste, on ne peut pas modifier ses éléments après création. Cette propriété le rend plus sûr et légèrement plus performant que la liste.

Création et accès

# Création
point    = (10.5, 20.3)
couleur  = (255, 128, 0)       # RGB orange
personne = ("Alice", 28, "Paris")
single   = (42,)                # ⚠ virgule obligatoire !
sans_par = 1, 2, 3             # tuple implicite

# Accès identique aux listes
print(point[0])       # 10.5
print(couleur[-1])    # 0

# Tentative de modification → TypeError !
# point[0] = 5    ← interdit

Unpacking — fonctionnalité clé

# Dépackaging simple
x, y = point                      # x=10.5, y=20.3
nom, age, ville = personne

# Échange de variables (élégant !)
a, b = 10, 20
a, b = b, a                       # swap sans variable temporaire

# Unpacking étendu avec *
premier, *reste = [1, 2, 3, 4, 5]
# premier=1, reste=[2,3,4,5]

*debut, dernier = [1, 2, 3, 4, 5]
# debut=[1,2,3,4], dernier=5

# Retourner plusieurs valeurs d'une fonction
def stats(data):
    return min(data), max(data), sum(data) / len(data)

mini, maxi, moy = stats([3, 1, 4, 1, 5, 9])

Méthodes disponibles

Le tuple n'ayant que 2 méthodes (immuabilité oblige) :

MéthodeDescriptionExemple
count(x)Nombre d'occurrences de x(1,2,1,3).count(1) # 2
index(x)Index de la 1ère occurrence de x(1,2,3).index(2) # 1
💡

En data analyse, les tuples sont utilisés comme noms de colonnes multi-niveaux dans pandas (MultiIndex), comme clés de dictionnaire (ils sont hashables, contrairement aux listes), et pour les coordonnées géographiques.

Conversion

t = tuple([1, 2, 3])   # liste  → tuple
l = list((1, 2, 3))    # tuple  → liste
t = tuple("hello")     # string → tuple : ('h','e','l','l','o')
03 — dict { }

Le Dictionnaire

Le dictionnaire associe des clés uniques à des valeurs. C'est la structure la plus proche du JSON, idéale pour modéliser des enregistrements structurés. Depuis Python 3.7, l'ordre d'insertion est garanti.

Création et accès

# Création
client = {
    "nom":    "Dupont",
    "prenom": "Marie",
    "age":    34,
    "actif":  True
}

# Depuis deux listes avec zip()
cles    = ["a", "b", "c"]
valeurs = [1, 2, 3]
d       = dict(zip(cles, valeurs))   # {"a":1,"b":2,"c":3}

# Accès aux valeurs
print(client["nom"])                  # "Dupont"
print(client.get("email"))            # None (pas d'erreur)
print(client.get("email", "N/A"))    # "N/A" — valeur par défaut

# Vérifier l'existence d'une clé
print("age" in client)               # True

Méthodes principales

MéthodeDescription
get(k, défaut)Retourne la valeur de k, ou défaut si absent (évite les KeyError)
keys()Vue de toutes les clés (itérable)
values()Vue de toutes les valeurs (itérable)
items()Vue des paires (clé, valeur) — très utilisé en boucles
update(d2)Fusionne d2 dans le dict (écrase les clés existantes)
pop(k, défaut)Retire et retourne la valeur de k
setdefault(k, v)Définit k=v seulement si k est absent, puis retourne la valeur
copy()Copie superficielle
clear()Vide le dictionnaire
dict.fromkeys(lst, v)Crée un dict depuis une liste de clés, avec valeur v

Itération

# Itérer sur les clés (défaut)
for cle in client:
    print(cle)

# Itérer sur les paires (le plus courant)
for cle, valeur in client.items():
    print(f"{cle:10} : {valeur}")

# Itérer sur les valeurs
for v in client.values():
    print(v)

Dict comprehension

# Syntaxe de base
carres = {x: x**2 for x in range(6)}
# {0:0, 1:1, 2:4, 3:9, 4:16, 5:25}

# Avec filtre
scores = {"Alice": 85, "Bob": 55, "Clara": 92}
admis  = {nom: s for nom, s in scores.items() if s >= 60}
# {"Alice":85, "Clara":92}

# Pattern data analyst : comptage de fréquences
data = ["A", "B", "A", "C", "B", "A", "B"]
freq = {}
for x in data:
    freq[x] = freq.get(x, 0) + 1
# {"A":3, "B":3, "C":1}

# Version one-liner avec collections.Counter
from collections import Counter
freq = Counter(data)   # Counter({"A":3,"B":3,"C":1})
✨

Fusion de dicts (Python 3.9+) : utilisez l'opérateur | pour fusionner deux dicts de façon élégante : d3 = d1 | d2. Pour Python < 3.9 : d3 = {**d1, **d2}.

04 — set { }

L'Ensemble (Set)

Un ensemble est une collection non ordonnée d'éléments uniques. Il est optimisé pour les tests d'appartenance (in) et les opérations ensemblistes (union, intersection, différence).

Création

# Création directe
a = {1, 2, 3, 4, 5}
b = {4, 5, 6, 7}

# Depuis une liste — dédoublonnage instantané !
doublons = [1, 2, 2, 3, 3, 3, 4]
unique   = set(doublons)    # {1, 2, 3, 4}

# ⚠ Important : le set vide s'écrit set(), pas {}
vide = set()               # {} crée un dict vide !

Méthodes de base

MéthodeDescription
add(x)Ajoute l'élément x (ignoré si déjà présent)
remove(x)Retire x — lève KeyError si absent
discard(x)Retire x — silencieux si absent (préférable)
pop()Retire et retourne un élément arbitraire
clear()Vide l'ensemble
copy()Copie de l'ensemble
len(s)Nombre d'éléments

Opérations ensemblistes

a = {1, 2, 3, 4, 5}
b = {4, 5, 6, 7}

# Union : tous les éléments
print(a | b)              # {1,2,3,4,5,6,7}
print(a.union(b))

# Intersection : éléments communs
print(a & b)              # {4,5}
print(a.intersection(b))

# Différence : dans a mais pas dans b
print(a - b)              # {1,2,3}
print(a.difference(b))

# Différence symétrique : non communs
print(a ^ b)              # {1,2,3,6,7}
print(a.symmetric_difference(b))

# Tests de relation
print({1,2}.issubset(a))   # True  ({1,2} ⊆ a)
print(a.issuperset({1,2})) # True  (a ⊇ {1,2})
print(a.isdisjoint(b))     # False (a et b ont des communs)

# Test d'appartenance — O(1), très rapide !
print(3 in a)             # True

Cas d'usage data analyst

# Dédoublonnage de données
ids_bruts = [101, 102, 101, 103, 102, 104]
ids_uniq  = list(set(ids_bruts))   # [101,102,103,104]

# Analyse de clientèle
clients_jan = {"Alice", "Bob", "Clara", "David"}
clients_fev = {"Bob", "David", "Emma", "Frank"}

fidelises  = clients_jan & clients_fev   # {"Bob","David"}
perdus     = clients_jan - clients_fev   # {"Alice","Clara"}
nouveaux   = clients_fev - clients_jan   # {"Emma","Frank"}
tous       = clients_jan | clients_fev   # tous les clients

# Vérification de colonnes requises dans un dataset
colonnes_requises = {"id", "date", "montant", "client_id"}
colonnes_fichier  = {"id", "date", "montant"}
manquantes        = colonnes_requises - colonnes_fichier
# {"client_id"}

Patterns Avancés

Imbrication de collections

# Liste de dictionnaires — très courant !
employes = [
    {"nom": "Alice", "dept": "IT",      "salaire": 45000},
    {"nom": "Bob",   "dept": "Finance", "salaire": 52000},
    {"nom": "Clara", "dept": "IT",      "salaire": 48000},
]

# Extraire tous les noms
noms   = [e["nom"] for e in employes]

# Filtrer le département IT
eq_it  = [e for e in employes if e["dept"] == "IT"]

# Trier par salaire décroissant
trie   = sorted(employes, key=lambda e: e["salaire"], reverse=True)

# Salaire moyen IT
sal_it = [e["salaire"] for e in employes if e["dept"] == "IT"]
moy    = sum(sal_it) / len(sal_it)   # 46500.0

Le module collections

from collections import Counter, defaultdict, OrderedDict, namedtuple

# Counter — compter des éléments
ventes  = ["A","B","A","C","B","A","B","B"]
c       = Counter(ventes)
print(c.most_common(2))        # [("B",4),("A",3)]

# defaultdict — dict avec valeur par défaut
par_dept = defaultdict(list)
for e in employes:
    par_dept[e["dept"]].append(e["nom"])
# {"IT":["Alice","Clara"], "Finance":["Bob"]}

# namedtuple — tuple avec attributs nommés
Point = namedtuple("Point", ["x", "y"])
p     = Point(10, 20)
print(p.x, p.y)                # 10 20
print(p[0], p[1])              # accès par index aussi

Exercices Pratiques

Des exercices progressifs orientés data analyst. Essayez de résoudre avant de voir la solution.

Exercice 1 — Liste

Nettoyage de notes

Vous avez une liste de notes : [12, 18, -5, 9, 20, 25, 14, 0]. Filtrez les notes valides (entre 0 et 20), calculez la moyenne, le min et le max.

▶ Voir la solution 
notes_brutes = [12, 18, -5, 9, 20, 25, 14, 0]

# Filtrage
notes   = [n for n in notes_brutes if 0 <= n <= 20]
# [12, 18, 9, 20, 14, 0]

# Statistiques
moyenne = sum(notes) / len(notes)   # 12.17
minimum = min(notes)                 # 0
maximum = max(notes)                 # 20

print(f"Notes valides : {notes}")
print(f"Moyenne : {moyenne:.2f} | Min : {minimum} | Max : {maximum}")
Exercice 2 — Dictionnaire

Inventaire produits

Créez un inventaire de produits avec nom, prix et stock. Affichez les produits en rupture (stock = 0), et calculez la valeur totale du stock (prix × quantité).

▶ Voir la solution 
inventaire = [
    {"nom": "Clavier",  "prix": 79.99,  "stock": 15},
    {"nom": "Souris",   "prix": 29.99,  "stock": 0},
    {"nom": "Écran",    "prix": 249.00, "stock": 8},
    {"nom": "Webcam",   "prix": 59.00,  "stock": 0},
]

# Produits en rupture
rupture = [p["nom"] for p in inventaire if p["stock"] == 0]
print(f"Rupture : {rupture}")        # ["Souris","Webcam"]

# Valeur totale
valeur  = sum(p["prix"] * p["stock"] for p in inventaire)
print(f"Valeur totale : {valeur:.2f} €")  # 3191.85 €
Exercice 3 — Set

Analyse d'audience

Vous avez les visiteurs de votre site sur 3 jours. Trouvez : les visiteurs fidèles (tous les jours), les visiteurs uniques totaux, et ceux qui sont venus uniquement le premier jour.

▶ Voir la solution 
jour1 = {"Alice", "Bob",   "Clara", "David"}
jour2 = {"Bob",   "Clara", "Emma",  "Frank"}
jour3 = {"Clara", "Bob",   "Grace", "Alice"}

fideles       = jour1 & jour2 & jour3   # {"Bob","Clara"}
tous          = jour1 | jour2 | jour3   # 7 visiteurs
j1_seulement  = jour1 - jour2 - jour3   # {"David"}

print(f"Fidèles 3 jours : {fideles}")
print(f"Visiteurs uniques : {len(tous)}")
print(f"Uniquement J1    : {j1_seulement}")
Exercice 4 — Combinaison

Analyse de ventes par catégorie

À partir d'une liste de transactions, calculez le chiffre d'affaires total par catégorie et identifiez la catégorie la plus rentable.

▶ Voir la solution 
transactions = [
    {"produit": "Laptop",  "cat": "Informatique", "montant": 1200},
    {"produit": "Chaise",  "cat": "Mobilier",     "montant": 250},
    {"produit": "Clavier", "cat": "Informatique", "montant": 80},
    {"produit": "Bureau",  "cat": "Mobilier",     "montant": 450},
    {"produit": "Écran",   "cat": "Informatique", "montant": 350},
]

# Agrégation par catégorie
ca_par_cat = {}
for t in transactions:
    cat = t["cat"]
    ca_par_cat[cat] = ca_par_cat.get(cat, 0) + t["montant"]
# {"Informatique":1630, "Mobilier":700}

# Catégorie la plus rentable
best = max(ca_par_cat, key=lambda k: ca_par_cat[k])
print(f"CA par catégorie : {ca_par_cat}")
print(f"Meilleure catégorie : {best} ({ca_par_cat[best]} €)")
